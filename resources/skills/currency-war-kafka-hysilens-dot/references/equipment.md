# 卡芙卡—海瑟音 DoT 装备

## 分配总则

```text
先保证卡芙卡拥有有效装备
→ 再把最高质量主输出装备交给海瑟音
→ 再补黑天鹅的终结技、能量或后台装备
→ 最后补椒丘、艾丝妲与功能角色
```

不要只背固定三件套。先检查：

- 装备是否匹配前台或后台位置；
- 是否增加行动或终结技频率；
- 是否提高全队对目标的伤害；
- 更适合多波次、小怪还是纯首领；
- 能否拆卸；
- 合成是否会让另一名核心失去多件有效装备。

## 卡芙卡

卡芙卡至少携带一件有效装备。她的机制与携带装备有关，即使后期转为后台，也不能把装备全部拆空。有条件时使用 2 至 3 件。

优先功能：

1. 速度和行动频率。
2. 攻击后施加脆弱或团队易伤。
3. 终结技启动与循环。
4. 前后台都能生效的强度。

有条件的高价值装备：

- **追逐星辰**：速度、幸运一击和攻击后脆弱，能放大海瑟音、黑天鹅及超激发。
- **反重力皮靴**：持续提高速度，增加卡芙卡行动与主动结算次数。
- **永动机**：支持开战终结技和后续终结技循环。

组合思路：

```text
速度/行动 + 团队易伤 + 终结技或通用强度
```

这些名称是高价值例子，不是唯一合法组合。

## 海瑟音

海瑟音固定前台，至少需要 2 件有效的输出、能量或速度装备。不要把只提供后台收益的装备当作核心。

优先功能：

1. 能量与快速启动。
2. 前台强度。
3. 速度和持续攻击能力。
4. 持续伤害或全局增伤。

有条件的高价值装备：

- **蓄能帆**：初始能量、持续回能和强度，帮助进入终结技循环。
- **白昼·光速螺旋桨**：速度、击破和强度；只有确实激活 3 昼之半神时才作为核心路线，不能为装备硬塞低质量成员。
- **杀红眼**：适合多波次和能稳定击杀小怪的战斗；纯首领开场不能立即获得完整收益。

组合思路：

```text
能量/启动 + 前台强度 + 速度或持续增伤
```

## 黑天鹅

黑天鹅固定后台，优先：

1. 开战终结技或终结技频率。
2. 后台强度。
3. 持续伤害增益。
4. 能帮助更快建立和维持奥迹的能量装备。

**永动机**适合快速开启终结技循环。**杀红眼**在多目标场景有价值，但同样依赖击杀，不是纯首领战的绝对必需。

组合思路：

```text
终结技启动 + 后台强度 + 持续伤害或能量
```

## 椒丘与艾丝妲

两者通常不占最高优先级：

- 先满足卡芙卡和海瑟音；
- 用剩余的速度、能量和辅助装备；
- 椒丘需要快速开终结技时，可以提高启动或能量优先级；
- 不让低星辅助占用海瑟音关键前台装备。

## 千冶·刃

千冶·刃可利用临时进阶装备，并在穿满三件时放大装备价值：

- 可以穿满三件“次优但有效”的装备；
- 不优先抢海瑟音的核心装备；
- 加入前先确认不会破坏 4 DoT 和生存；
- 检查星核猎手的承载者判定，避免装备数量导致错误角色成为主承载者。

该模块的最优装备分配仍属于待验证结论。
