# 白厄直伤与反甲阵容核心

## 白厄机制

白厄是 3 费前台输出，独立羁绊为救世主，主要刷新等级为 7。

### 我独自战斗

白厄能够获得前台队友的羁绊效果和属性。评估队友时需要同时检查前台羁绊、属性和星级。没有关键档位差异时，高星通用前台可能比一星理论挂件更有价值。

### 背负希望

白厄积累火种并变身为卡厄斯兰那，变身额外回合速度受白厄速度影响。其他前台角色每个星级会提高白厄前台强度，且白厄星级越高，转化越强。

阵容价值取决于：

```text
白厄星级
+ 前台队友总星级
+ 前台队友的有效羁绊和属性
```

## 直伤白厄

依靠白厄变身、额外回合、队友属性/羁绊继承、星级转化以及速度和伤害装备。

优点：

- 装备要求较宽松；
- 不等待敌人攻击；
- 没有反甲时也能运行；
- 适合前期过渡和低攻击频率首领。

弱点是后期需要更高星白厄、前台队友和额外乘区。

## 反甲白厄

以牙还牙甲通过装备者防御建立前台护盾，并在装备者受击时根据受击前护盾量造成物理反伤。反伤不能消灭敌方。

核心循环：

```text
高防御角色建立护盾
→ 敌人攻击反甲装备者
→ 按受击前护盾量触发反伤
→ 多次受击或多件反甲累计伤害
→ 白厄、副输出、追击或列车完成收尾
```

反甲主要看防御、护盾、受击前护盾量、反甲数量、攻击频率、攻击目标和护盾维持，不以攻击力为主要倍率。

上限高是因为防御、护盾、多件装备和高频攻击可以形成大量结算；下限低是因为缺少反甲/护盾、敌人不攻击装备者、削盾/绕盾、低攻击频率或缺少收尾都会让体系失效。

## 白厄星级

- 一星：可作直伤过渡；反甲、护盾和高星前台齐全时也能运行。
- 二星：最高难度的稳定目标，提高自身、星级转化、变身和混合路线稳定性。
- 三星：条件式上限。只有明显数量进度、7 级还有其他 3 费收益、阵容已成型、生命健康且有止损时追。

需要人口、反甲不足、护盾和收尾未解决，或下一战急需确定战力时，不追三星。

## 功能模块

### 护盾核心

评估防御、护盾量和频率、星级、前台羁绊、受击概率和生存。砂金是候选之一，但不是唯一答案；高星的其他护盾角色可能更优。

### 高星前台

排序：

1. 护盾、防御或生存；
2. 星级与属性；
3. 关键羁绊；
4. 收尾能力；
5. 是否只是低质量挂件。

### 收尾

反伤不能击杀，必须保留白厄主动攻击、高星副输出、追击、列车、群体或其他可靠伤害。

## 阵容形态

### 前期直伤白厄

```text
白厄 + 高星前台 + 生存 + 通用副输出/辅助
```

没有反甲时正常穿直伤装备，不空等组件。

### 一件反甲过渡

给白厄或护盾核心一件反甲，观察受击目标、护盾维持、反伤占比、收尾和词缀适配。

### 两件反甲稳定形态

通常意味着路线明确。后续优先防御、护盾、白厄二星、高星前台和收尾。

### 三件反甲高上限

仅在两件已稳定、第三件组件接近、白厄和护盾成型、首领适配且有可靠收尾时追求。

### 3 巡海游侠 + 2 列车同行

这是可用攻略骨架，不是唯一名单。根据星级、星徽、护盾、环境、首领和收尾动态替换。
