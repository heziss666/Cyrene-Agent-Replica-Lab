# 白厄与以牙还牙甲装备

## 以牙还牙甲

合成组件：

```text
幸运星 + 量产型装甲
```

装备提供幸运一击、伤害减免和护盾强度。战斗开始为全体前台提供基于装备者防御的护盾；装备者受到攻击时，根据受击前护盾量对攻击者造成物理反伤。反伤不能消灭敌人。

数量意义：

- **一件**：验证路线，观察受击、护盾和反伤占比。
- **两件**：稳定形态，可以围绕防御、护盾和受击目标继续建设。
- **三件**：条件式高上限，仅在组件、敌人、护盾和收尾都满足时追求。

合成前检查组件是否被核心使用、拆装能力、装备者、防御、护盾来源、下一敌人攻击方式和收尾。

## 装备者选择

可考虑白厄、高防御护盾角色、更容易受击的前台或能稳定维持护盾的单位。

比较：

1. 受击概率；
2. 防御；
3. 护盾稳定性；
4. 生存能力；
5. 反伤后的收尾；
6. 多件装备集中或分散的实际收益。

不要假设所有反甲必须穿在白厄身上。

## 白厄直伤装备

没有反甲或只处于混合路线时，白厄仍需要主动输出。优先：

- 前台强度；
- 速度和额外回合启动；
- 初始能量；
- 伤害增幅；
- 按星级成长；
- 生存。

高周波电锯、热血沸腾拳、杀红眼等属于条件式直伤候选。具体效果以结构化数据基线为准。

## 防御与护盾

反甲优先功能：

```text
防御 + 护盾强度 + 伤害减免 + 开战护盾 + 持续补盾
```

攻击力不是反甲主要倍率。不要为第三件反甲拆掉唯一护盾装备、白厄必要主动输出装备或最终首领前更确定的战力。

## 常见错误

- 一件反甲就当作终局成型。
- 不检查敌人是否攻击装备者。
- 不保留主动收尾。
- 只堆攻击力解释反伤。
- 护盾经常被清空仍继续追加反甲。
- 剩余节点很少时从零追第三件。
