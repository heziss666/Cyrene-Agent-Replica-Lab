# 经济、升级与刷新

## 适用前提

- 当前玩法为 4.4 标准博弈。
- 当前问题涉及金币、经验、等级、商店刷新、角色升星或下一战压力。
- 利息、升级成本、刷新成本和商店概率必须从 GameState 或结构化规则查询。
- 本文件不提供固定金币线，也不把某个等级写成所有阵容的统一刷新等级。

## 总体决策顺序

```text
先判断下一战风险
→ 再找可以确定获得的即时战力
→ 再比较升级与刷新的边际收益
→ 最后计算长期经济损失
```

下一战很可能失败时，长期经济不应自动压过即时战力。

阵容稳定且没有确定提升时，保留经济和路线弹性更有价值。

## 五类金币用途

金币可转化为：

1. 购买当前商店角色；
2. 完成角色升星；
3. 购买经验并升级团队规模；
4. 刷新商店；
5. 保留经济，为后续节点提供更多操作空间。

每次消费前询问：

```text
这笔金币带来什么确定变化？
何时生效？
能否改变下一战结果？
如果失败，损失是什么？
如果不花，后续增加什么选择权？
```

## 更偏向升级的条件

- 升级后可以立即上一个有价值的角色；
- 新位置能完成关键羁绊；
- 新位置能补生存、收尾或核心辅助；
- 当前目标费用更适合更高等级商店；
- 当前等级没有关键对子；
- 当前阵容能通过下一战；
- 剩余节点足以利用更高等级的长期收益；
- 备战席已有明确可上场单位。

如果升级后新位置为空，或只能放低质量挂件，升级的即时收益下降。

## 更偏向刷新的条件

- 商店、场上与备战席中存在“一张即可升星”的核心；
- 同时有多个高价值对子；
- 当前等级更适合目标费用；
- 下一节点是高风险战斗或首领；
- 生命压力较高；
- 升级不能立即增加有效战力；
- 当前人口已足够容纳核心阵容；
- 免费刷新或投资效果降低了刷新代价。

刷新前必须明确：

```text
找哪名角色
完成哪次升星
达到哪个羁绊档位
最多承受多少次无提升
触发什么条件立即停止
```

## 更偏向保留经济的条件

- 生命健康；
- 下一节点风险较低；
- 最近战斗有明显余量；
- 当前没有可完成的核心对子；
- 路线尚未确认；
- 商店没有立即有价值的购买；
- 升级后没有合适角色填入；
- 后续即将进入更适合目标费用的阶段；
- 投资环境或策略将在未来提高消费效率。

“保留经济”不等于不操作。仍应完成：

- 免费站位调整；
- 免费装备调整；
- 必买的确定性升星；
- 低成本高价值角色；
- 锁定明确的高价值商店。

## 生命压力如何影响经济

生命值既是失败资源，也影响通关评价。

提高止损权重的信号：

- 最近一战失败；
- 连续多战接近行动值上限；
- 前台过早倒下；
- 下一节点是首领；
- 下一遭遇风险较高；
- 当前生命接近评价边界；
- 核心阵容仍只有一星；
- 有明确可完成的升星或羁绊。

生命压力高时，应优先购买确定提升，而不是为理论终局无目标刷新。

## 即时战力与长期上限

### 即时战力

本节点或下一战就发挥作用：

- 完成升星；
- 增加有效上阵位；
- 完成羁绊档位；
- 将装备交给正确角色；
- 修正前后台；
- 加入生存或首领对策；
- 用高星过渡替换无效挂件。

### 长期上限

需要更多节点、角色或组件才兑现：

- 从零追三星核心；
- 提前购买无法上场的高费核心；
- 为未来羁绊保留多个一星挂件；
- 选择长延迟投资；
- 升级后等待未来高费商店；
- 收集尚不能合成的装备组件。

高风险节点前，即时战力权重上升。

生命健康、阵容稳定且剩余节点多时，长期上限权重上升。

## 核心对子与升星进度

购买价值提高：

- 购买后立即升星；
- 购买后形成明确对子；
- 同时补关键羁绊；
- 角色已有合适装备；
- 角色是阵容 Skill 指定的发动机或生存核心。

购买价值降低：

- 只是第一张未来高费核心；
- 当前无法上场；
- 会挤占备战席；
- 需要大量后续刷新才有意义；
- 另一条路线已有更成熟进度。

## 刷新止损

任何“递牌”建议都要包含停止条件。

可采用：

```text
完成目标升星即停
完成关键羁绊即停
下一战达到稳定标准即停
升级所需资源必须保留
连续刷新没有形成新对子时重新评估
生命压力解除后停止追加投入
```

不得只说：

```text
继续D
梭哈
刷到出为止
```

除非最终首领前没有其他可转化战力的方式，并且必须明确这是高风险策略。

## 各阶段经济目标

### 第一位面

- 依靠自然刷新、奖励与低成本角色建立过渡；
- 不为尚未确认的最终阵容频繁手动刷新；
- 购买能立即升星或形成低成本羁绊的角色；
- 保留至少两条路线的可能性；
- 第一位面首领前战力不足时允许有限止损。

### 第二位面

- 根据核心角色进度、装备和羁绊确认路线；
- 本节点只选一个主任务：升级、刷新或保经济；
- 完成中期关键升星；
- 不同时追多个互相竞争的终局目标；
- 第二位面首领前确保阵容已从过渡转为稳定。

### 第三位面

- 用经济提高上限和最终首领胜率；
- 比较三星进度与人口收益；
- 完成高费功能角色、关键装备或对策；
- 越接近最终首领，延迟收益价值越低；
- 不为无法完成的三星目标浪费全部资源。

## 路线未确定时

尚未确定阵容，不代表什么都不买。

优先购买：

- 可自然二星的低中费角色；
- 通用生存；
- 多体系可用辅助；
- 可承接装备的过渡输出；
- 同时属于多条潜在路线的成员；
- 能保持连胜并减少生命损失的单位。

不要因为最终攻略名单没有某角色，就卖掉当前最强的二星过渡。

## 需要结合 JSON 查询的事实

- 当前升级成本和经验；
- 当前等级商店费用概率；
- 角色费用与站位；
- 投资环境与策略的经济修正；
- 商店锁定、禁止或替换规则；
- 免费刷新次数；
- 羁绊经济效果。

## 不可当作硬规则的经验

- 所有阵容在固定金币数刷新；
- 任何情况下都先升人口；
- 所有三费核心在同一等级刷新；
- 生命健康时永远不能刷新；
- 低生命时一定花光；
- 高费一星一定强于已成型二星低费。
