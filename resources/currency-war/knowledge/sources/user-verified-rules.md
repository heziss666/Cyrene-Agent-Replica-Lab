# Standard Gambit Fixed Node Sequence

This record preserves the fixed standard-gambit node order confirmed by the project owner during requirements review. It is a local canonical rule record, not an inference from the RAG implementation plan.

- Plane 1: 1-1 reward, 1-2 reward, choose investment strategy, 1-3 battle, 1-4 battle, 1-5 supply, 1-6 battle, 1-7 encounter, 1-8 reward, 1-9 boss.
- Plane 2: 2-1 battle, choose investment strategy, 2-2 battle, 2-3 supply, 2-4 battle, 2-5 encounter, 2-6 reward, 2-7 boss.
- Plane 3 uses the same node-type distribution as Plane 2.

Use this record only for fixed node-order claims. Specific random rewards, enemies, shops, investment effects, and combat outcomes are not covered by it.
